# Todo list backend app

## Description

Todo list backend app is a backend application to create api's for todolist using express and restful api's so that the user can add, edit, view and delete the tasks.

## Tech Stack

- **Express** with Typescript
- Node.js (for execution)

## Clone Project

- Clone the project and change directory to todo-list-backend-app

```bash
git clone https://github.com/ChandanaThoti/todo-list-backend-app.git
cd todo-list-backend-app
```

## Installation

```bash
npm install
```

## Start Project

1. **Start the application** using:

```bash
   npm run build      # Build the application
```

```bash
   npm start       # Start the application
```
